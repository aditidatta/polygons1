lambdas

Lambdas are one of those things which are very confusing at first but once you get it you will see how cool it is. Lambdas are basically functions, more appropriately anonymous functions. They don't belong to any specific class and can be passed around like objects. Here I am going to show how to write lambdas in Java and Python. The same concept is shared by many other languages like Javascript, C++, C#, Swift.
