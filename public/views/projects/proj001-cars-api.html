Cars API for Craigslist

Provides a REST API to Craiglist's used cars' postings.

Crawler:

Crawls the website for postings, scrapes and stores only the make, model, price and a few other properties and the url to the original posting.

Query API:

The REST layer, it's job is to parse queries and fetching data from DTO layer or Cache.

Data Source:

Cache and MySQL Database

Tech Stack:

Java, JUnit, Spring MVC, Spring Boot, JPA, Hibernate, Jackson, Jenkins maven, Digital Ocean
